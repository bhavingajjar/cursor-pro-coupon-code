## Cursor Cloud specific instructions

This repository is **documentation-only**. The only tracked file is `README.md`, an
SEO/referral landing page for a Cursor Pro discount code. There is:

- No source code, application, backend, or frontend.
- No package manager, dependency manifest, or lockfile.
- No build, lint, or test tooling, and no services to run.

### Working in this repo

- "Development" means editing `README.md` and committing with git. There is nothing to install.
- The "product" is the README as rendered by GitHub. To preview rendering locally, use a
  GitHub-flavored Markdown previewer (e.g. `grip README.md`, which needs network for GitHub's
  rendering API). This is only a preview aid and is **not** a project dependency — do not add it
  to any startup/update script.
